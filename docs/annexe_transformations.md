# Annexe — Transformations

Auteur : Fabien Hummel

Ce document présente les fonctions de préparation des données.

- clean_text : nettoie les textes.
- extract_number_available : extrait le stock disponible.
- convert_rating_to_number : convertit la note en nombre.
- transform_book : prépare un livre pour le CSV.
- transform_books : prépare une liste de livres pour le CSV.
